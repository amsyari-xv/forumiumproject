## About Forumium

A Web Based Forum Platform for Students
ACE6173 Software Engineering Assignment

## Tech Stack
- Flarum (PHP, MySQL)
- Nginx
- Google Cloud VM (Ubuntu)
- GitHub for version control

## Deployment
- Hosted on Google Cloud VM
- Nginx + PHP-FPM
- Database managed via MySQL

## Version Control Strategy
- Flarum core excluded (open source)
- Runtime files ignored
- Sensitive configs excluded

## Features
- User registration
- Login / logout
- Discussions & replies
- Admin moderation
